== Description ==

WP-Obscure will change the content of a post in the eyes of a search engine spider, but
not the eyes of the reader. It will replace various characters with ascii values and
add hidden html tags inside the post to change the structure.

== Installation ==

1. Upload `wp-obscure` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You're all set... all posts will automatically be obscured.
